# Task 01 - Barcelona Shared Waste System Research

## Goal
Document how Barcelona's shared container waste system works and what evidence
exists regarding efficiency, cleanliness, labor, and costs. Synthesize findings
into a written summary that informs the assumptions used in Tasks 04 and 05.

## Owner
Scribe (supporting: Lead)

## Dependencies
None -- this task can start immediately.

## Inputs
- `research/Shared Containerized Waste Collection_ Lessons from Barcelona and Beyond.pdf`
  (already present in repo -- primary source)
- Secondary sources: academic papers, city reports, news articles (open web)

## Outputs
- `research/barcelona_shared_waste.md` -- written summary (>=1,000 words)

## Required Sections in Output
1. **System Overview** -- how the Barcelona system works (container types, semi-underground
   vs surface, collection vehicles, separate waste streams)
2. **Key Metrics** -- quantitative data on cost, labor, cleanliness, recycling rates
   where available
3. **Evidence Quality** -- distinguish peer-reviewed evidence from city marketing claims
4. **Drawbacks & Controversies** -- accessibility concerns, upfront capital, community
   acceptance issues, disability access
5. **Assumptions Derived** -- explicit table of numbers this research supports that
   feed into Tasks 04 and 05 (e.g., container size, utilization rate, collection frequency)
6. **Transferability to DC** -- what adapts, what doesn't, what's unknown
7. **What We Know / Don't Know** -- honest gap assessment
8. **Sources** -- full citation list with URLs/DOIs where available

## Constraints
- Distinguish evidence-backed claims from aspirational claims
- Flag any numbers used as assumptions in later tasks explicitly
- Do not fabricate citations -- only cite sources that can be verified

## Acceptance Criteria
- [ ] `research/barcelona_shared_waste.md` exists with all 8 required sections
- [ ] >=3 credible sources cited (with URLs or DOIs)
- [ ] All assumption values feeding into Tasks 04/05 listed in "Assumptions Derived"
      table with source attribution
- [ ] "What We Know / Don't Know" section is present and substantive
- [ ] File is valid Markdown and renders cleanly
